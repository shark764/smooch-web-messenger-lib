Prod Web Chat

Add ?tenantId=your-tenant-id&appId=your-app-id to the URL for the chat widget in this page to work.
For region, add region=eu-1 to the URL params.
For proactive chat, add proactiveChatMessage=Hello%21%20How%20are%20you%20doing%3F% to the URL params. To add a timer to automatically show the proactive chat, add proactiveChatTimer=2 to the URL params.
For custom text, add &prechatCaptureGreetingText=supdawg&prechatCaptureConfirmationText=neato to the URL params.
